Accessibilité

Pâtissons s'engage à rendre son site accessible à toutes et à tous, conformément aux recommandations des WCAG 2.1 niveau AA.

Mesures prises

  • Structure HTML sémantique avec les balises appropriées (header, main, nav, footer, article)
  • Navigation au clavier possible sur l'ensemble du site
  • Attributs aria-label sur les éléments interactifs
  • Textes alternatifs sur toutes les images
  • Contrastes de couleurs respectant les normes WCAG AA
  • Mise en page responsive adaptée à tous les écrans
  • Fil d'ariane sur toutes les pages pour faciliter la navigation

Limites connues

Ce site est un projet en cours de développement. Certaines fonctionnalités peuvent ne pas être encore totalement accessibles. Nous travaillons à améliorer continuellement l'expérience de tous les utilisateurs.

Nous contacter

Si vous rencontrez une difficulté d'accessibilité sur notre site, n'hésitez pas à nous le signaler via notre page de contact. Nous ferons notre possible pour vous apporter une réponse dans les meilleurs délais.

Technologies utilisées

  • HTML5 sémantique
  • CSS3 avec media queries
  • JavaScript vanilla

Outils de test

  • Google Lighthouse
  • WAVE — Web Accessibility Evaluation Tool
  • axe DevTools
